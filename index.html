What is Pithos?

Pithos is an online Python IDE, written entirely within JavaScript. It is highly extendible, and supports the following:

  • Over 50+ libraries
  • Local file cache
  • Filecodes and data saving
  • And more!
Pithos also has it's very own PyCode extension, which allows you to parse Python Code embedded in a webpage, a useful Python Shell that you can access in your web browser, File I/O support on your own computer, a JavaScript console, and all kinds of other cool additions to your web browser. More information about PyCode can be found here.

PyCode

What is PyCode, exactly?

PyCode is a combination of Developer Tools for JavaScript and Developer Tools for Python. It allows you to run and embed Python code into a webpage, like this: 

				<%python%
					<p>This was printed with Python!<p>
				%>
Running that code would display:

This was printed with Python!

It also contains a Python Shell, JavaScript shell, File I/O manager(for when the code tries to use it), and all kinds of other features. PyCode also works seamlessly with Pithos, allowing it to read files outside of the virtual file file system.
